What is Control Flow?

code executes in a top-down manner, unless the code control changes its course when it meets with other structures such as: Loops, Conditionals, Functions

Loops

Loops are a conditional statement that repeats the given task until the given condition is met. Loop statements are also known as iteration statements. There are three types of loops

  • For
  • While
  • Do while
  • For Loop
The for loop is one of the most commonly used loop structures in most programming languages, not just JavaScript. It is also simple to use.

Conditionals

the programmer writes several codes which will be executed if the condition is met. If the condition is not met, then another statement is executed.

  • if
  • Else if
  • if...else
  • Switch
  • if Statement

Funtions

When you write a specific block of code for a certain purpose, that code will be needed in various places of the code, which is a typical programming issue. Therefore, writing the same code repeatedly is not a wise option, which is why functions exist. Functions eliminate the need for repeating the same code. All you will need to do is write a function then use it whenever you want at any part of the program.

What's DOM?

magine your website is like a house. The DOM (Document Object Model) is like the blueprint of the house. It organizes all the parts of the house (like rooms, doors, windows, etc.) so that you can see how everything is connected. The DOM is how we can interact with the parts of the webpage using code! The DOM represents the structure of a webpage. It lets us access, change, add, or remove parts of the webpage using JavaScript.

Array

An array is like a list of items. The items inside are ordered, and each item has a position number (called an index), starting from 0.

Objects

An object is like a box that can store different pieces of information (called properties) with a name and a value. The properties don’t have an order like in arrays, but you access them using the name of the property.

Array of objects

An array of objects means that the array holds multiple objects, and each object inside the array can have different properties. 

let people = [
            { name: "Alice", age: 25, city: "New York" },
            { name: "Bob", age: 30, city: "London" },
            { name: "Charlie", age: 22, city: "Paris" }
          ];
for (let i = 0; i < people.length; i++) {
            console.log(people[i].name + " is " + people[i].age + " years old and lives in " + people[i].city);
          }

Funtions

A function is a reusable block of code designed to perform a specific task. Instead of writing the same code over and over again, you can define a function once and use it whenever you need it by "calling" the function.

// Defining a function
          function sayHello(name) {
            console.log("Hello, " + name + "!");
          }
          
          // Calling the function
          sayHello("Alice"); // Output: Hello, Alice!
          sayHello("Bob");   // Output: Hello, Bob!
